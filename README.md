# ✨ ⚔️ Projeto: A Cavaleira Branca

Este repositório contém a ficha mística de **Mira Haverinen**, uma fada guerreira da Corte Luminae. O projeto foi desenvolvido para capturar a essência "fae" e heróica da personagem, unindo funcionalidade de RPG com uma narrativa visual imersiva.

## 🧚‍♀️ Sobre Mira Haverinen

Conhecida como **A Cavaleira Branca**, Mira é uma fada que desafiou as expectativas da realeza para viver sua verdade. Abençoada pela Rainha Solastra e imbuída da coragem das Guardiãs, ela vaga pelo mundo de Everfree, carregando o peso de um amor antigo e o brilho de sua armadura impecável.

> *"Você seria capaz de se juntar a mim?"*

---

## 🔮 Funcionalidades da Ficha

O site foi projetado com uma interface intuitiva e elementos que evocam o pó de pirlimpimpim e o aço frio:

* **Painel de Atributos:** Visualização clara das capacidades físicas e mentais (com foco na sua agilidade élfica).
* **Inventário:** Listagem de itens raros, desde armas pesadas como a *Claymore* até artefatos curiosos como a *Concha Colorida*.
* **Sistema de recrutamento para membros:** Sistema interativo de aliança para recrutamento de novos aventureiros para o grupo.

## 🍃 Execução

1. Clone o repositório:
```bash
git clone https://github.com/usuario/ficha-de-rpg.git

```


2. Instale as dependências necessárias:
```bash
npm install

```


3. Inicie o projeto local:

```bash
   npm run dev

```

4. Abra o navegador em `http://localhost:3000`.
